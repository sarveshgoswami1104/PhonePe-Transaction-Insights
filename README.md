# 📊 PhonePe Transaction Insights

An end-to-end Machine Learning and Data Analytics project built using the **PhonePe Pulse** dataset. This project extracts complex nested JSON data, transforms it into a structured SQL database, performs exploratory data analysis, trains a Machine Learning model to predict transaction amounts, and visualizes insights through an interactive Streamlit dashboard.

---

## 🚀 Project Overview

The objective of this project is to analyze PhonePe digital payment transactions across different states, districts, years, and transaction categories to generate meaningful business insights. The project follows a complete Data Science workflow, including data extraction, preprocessing, SQL integration, visualization, predictive modeling, and dashboard development.

---

## ✨ Features

- Automated ETL pipeline for nested JSON datasets
- Data extraction from the PhonePe Pulse GitHub repository
- Data cleaning and preprocessing
- SQLite database integration

## 📊 Machine Learning Workflow

1. Extract JSON data from the PhonePe Pulse dataset
2. Transform nested data into structured DataFrames
3. Load processed data into SQLite
4. Perform SQL queries for business analysis
5. Conduct Exploratory Data Analysis (EDA)
6. Engineer features for machine learning
7. Train and evaluate a Random Forest Regression model
8. Save the trained model
9. Build an interactive Streamlit dashboard

## ▶️ How to Run

### Clone the Repository

```bash
git clone https://github.com/sarveshgoswami1104/PhonePe-Transaction-Insights.git
```

### Navigate to Project

```bash
cd PhonePe-Transaction-Insights
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Launch Jupyter Notebook

```bash
jupyter notebook
```

### Run Streamlit Dashboard

```bash
streamlit run app.py
```
